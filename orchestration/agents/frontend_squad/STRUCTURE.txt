Workflows Prédéfinis
====================

1. design_to_code
   Designer: Create design system
        ↓
   Architect: Design component architecture
        ↓
   Developer: Implement components

2. component_creation
   Designer: Design UI/interactions
        ↓
   Architect: Define API/variants
        ↓
   Developer: Implement code
        ↓
   Architect: Create Storybook docs

3. design_system_creation
   Designer: Define design tokens
        ↓
   Architect: Structure library
        ↓
   Developer: Implement base components

Installation Rapide
===================

1. Installer les dépendances:
   pip install httpx python-dotenv pytest pytest-asyncio

2. Configurer la clé API:
   echo "OPENROUTER_API_KEY=your-key" >> .env

3. Tester l'installation:
   python -c "from orchestration.agents.frontend_squad import get_squad_info; print(get_squad_info())"

4. Exécuter les tests:
   pytest orchestration/agents/frontend_squad/test_frontend_squad.py -v

5. Lancer les exemples:
   python orchestration/agents/frontend_squad/example_usage.py
